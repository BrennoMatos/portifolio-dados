# Projetos de Ciência de Dados

Esta seção reúne projetos práticos desenvolvidos para demonstrar conhecimentos em análise de dados, machine learning e resolução de problemas de negócio.

## Projetos

### 1. Previsão de Custos de Manufatura

**Objetivo:** prever custos de produção com base no volume de unidades vendidas.

**Tecnologias:**

* Python
* Pandas
* Scikit-Learn
* Matplotlib

**Competências demonstradas:**

* Análise exploratória de dados
* Estatística descritiva
* Correlação entre variáveis
* Modelagem preditiva
* Avaliação de modelos

📂 Projeto: `previsao-custos-manufatura`
